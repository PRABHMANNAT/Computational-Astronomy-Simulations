# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus criticism on ideas and implementation details.
- Prefer clear explanations and reproducible examples.
- Respect scientific uncertainty and cite assumptions where relevant.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Personal attacks.
- Publishing private information without permission.
- Repeated disruption of project discussions.

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation when behavior is inconsistent with this code of conduct.
