# Exoplanet Atmosphere Spectrum

Status: planned.

## Scope

Visualize simplified transmission spectra and molecular absorption bands for exoplanet atmospheres.

## Planned Architecture

- `src/physics` - absorption depth and wavelength band helpers.
- `src/simulation` - atmosphere parameter state.
- `src/components` - spectrum chart and controls.
- `src/data` - molecular bands and example atmospheres.
