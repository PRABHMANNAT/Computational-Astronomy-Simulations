# Cosmic Web Formation

Status: planned.

## Scope

Visualize simplified matter clustering into filaments, nodes, and voids.

## Planned Architecture

- `src/physics` - simplified clustering and expansion helpers.
- `src/simulation` - particle field state.
- `src/components` - 3D cosmic web renderer.
- `src/data` - seeded matter distributions.
