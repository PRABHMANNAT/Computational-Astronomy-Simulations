# Habitable Zone Explorer

Status: planned.

## Scope

Explore approximate habitable-zone boundaries for stars of different luminosities.

## Planned Architecture

- `src/physics` - habitable-zone boundary calculations.
- `src/simulation` - stellar and orbital state.
- `src/components` - zone chart and controls.
- `src/data` - stellar type presets.
